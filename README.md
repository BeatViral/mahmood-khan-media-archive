# Mahmood Khan Media Archive

Standalone static archive page for:

**Mahmood Khan Media Archive**  
Artist, Founder & Audio Technology Builder

Open `index.html` in a browser to preview it locally.

## Editing Links

Add or replace entries in `archive-data.js`.

Each signal supports:

- `title`
- `source`
- `year`
- `category`
- `region`
- `priority`
- `description`
- `url`

Leave `url` blank for planned links; the card will show `Link Pending`.
